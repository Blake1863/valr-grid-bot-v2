## Key Learnings

1. **Neutral mode hedging:** Position + all orders should = 0. When position exists, place extra orders on opposite side.

2. **TPSL rebuild optimization:** Only rebuild when avg entry price changes, not on every position update. Better yet, only rebuild on actual fills.

3. **Grid sizing at 10x:** With 10x leverage and symmetric grid, when one side fills you get ~5x effective exposure (half the grid becomes position).

4. **Self-limiting design:** Don't need separate max position constraint if grid sizing is correct.

5. **Restart with position:** Only place hedging orders, don't try to add to position (insufficient balance).
